Code Book 
=======================================================

This code book was submitted after the deadline, as part of a peer assessment project for Coursera module "Getting and cleaning data"

The tidy dataset contains 6 files, one per labeled activity. Each file contains 30 records, one per observational unit (definition of *tidy*).

All variables 3 to 68 are aggregate measurements (either mean or standard deviation). The variable names were bulit to be self explicit so we don't feel the need to describe further.

[1] "subject_id" : integer (1..30) identifying an observational unit, that is a subject holding a smartphone part of the study
 
[2] "activity_label" : the activity under wich measurements in this dataset were made (one of LAYING, SITTING, STANDING, WALKING, WALKING_UPSTAIRS, WALKING_DOWNSTAIRS)
